# spaaccesiblefabiancaranton-
## Descripcion de la actividad
Esta actividad busca saber crear wireframes de baja fidelidad, como ejemplo practico, se diseño una interfaz basica de comercio electrónico, diseñada bajo paradigmas de Inclusión  El objetivo central es conocer como podemos plasmar por medio de diseños basicos en papel, ideas de diseño e implementar procesos de accesibilidad pertinentes, menus accesibles para diferentes usuarios.
